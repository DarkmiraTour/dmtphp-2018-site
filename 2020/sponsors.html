Patrocinadores Ouro

Blackfire.io

O Blackfire.io permite que os desenvolvedores PHP verifiquem e melhorem continuamente o desempenho dos seus aplicativos no ambiente de desenvolvimento, teste, integração e produção. Torna possível detalhar ao nível de call de função / método para entender e corrigir problemas de desempenho. Sua ampla variedade de opções de automação faz com que seja um alívio adicioná-lo a um workflow de desenvolvimento e teste.

O Blackfire Player, um poderoso Open Source Web Crawling, Web Tester e Web Scraper permite que os times configurem um pipeline de testes totalmente automatizado.
